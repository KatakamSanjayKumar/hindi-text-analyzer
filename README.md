# Hindi Text Analyzer

A Gradio-based Python application to analyze Hindi text using Unicode-aware processing.

## Features
- Word count
- Raw character length
- Visible Hindi units (Aksharas)
- Matra count
- Sign count
- Word-level breakdown

## Tech Stack
- Python
- Gradio
- Regex
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
